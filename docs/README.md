# Documentation Notes

This directory contains project-level method notes and portfolio summaries:

- `project_summary.md`: scope, implemented workflow, synthetic results, and
  future extensions.
- `rotational_diagram_assumptions.md`: equations, assumptions, and failure
  modes for the LTE demonstration.
- `cv_bullets.md`: concise portfolio and CV wording.

The source notebooks contain the current explanations of FITS/WCS handling,
moment maps, Gaussian line fitting, and measured quantities. RADEX-style
non-LTE grid comparison remains planned and is not implemented.
